- 只有18.8%的期望关键词被找到
- 即使找到了一些结果，相关性也不够强

#### 可能原因
1. **评分机制问题**: 综合评分可能偏向某些维度（如时间、来源），而忽略了相关性
2. **数据质量问题**: 存储的内容可能不够完整或准确
3. **embedding模型问题**: 语义检索可能没有很好地捕捉到关键词的语义
4. **搜索策略问题**: 语义检索和关键词检索的权重分配不合理

### 改进建议
1. **调整评分权重**:
   ```python
   # 建议的权重分配
   - 相关性: 45% (当前35%) ⬆️
   - 匹配度: 25% (当前15%) ⬆️
   - 时间: 15% (当前20%) ⬇️
   - 重要性: 10% (当前15%) ⬇️
   - 来源权重: 5% (当前15%) ⬇️
   ```

2. **增强关键词匹配**:
   - 提高关键词匹配的权重
   - 实现更智能的关键词匹配（如部分匹配、同义词匹配）

3. **优化评分阈值**:
   - 默认min_score从0.5降低到0.3
   - 允许用户自定义阈值

4. **改进数据质量**:
   - 预存更多高质量的测试数据
   - 确保存储的内容包含完整的关键词
